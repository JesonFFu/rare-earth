# Rare-Earth Photonic Grating Designs

This repository contains COMSOL/Python design work for rare-earth and silicon nitride photonic grating/quasi-BIC simulations.

## Repository Contents

- `sin980_qbic_*` and `sin451_qbic_*`: design iterations, scripts, scan results, plots, and summaries.
- `sin980_451_qbic_*`: dual-wavelength design exploration.
- `comsol_bic_design_conversation_summary.tex`: project summary source.

## Large Files

COMSOL model files (`*.mph`), run logs, cache files, PDFs, and presentation binaries are ignored by default to keep the Git repository lightweight.

If a specific COMSOL model should be versioned, add it deliberately with Git LFS instead of regular Git.
